### 说明：

1、登录支持账号密码登录与人脸识别登录。

2、可通过门禁提供的验证码来查看来访人员信息与门禁开启授权。

3、进出情况通过时间轴的形式查看来访进历史记录。

4、注册成员管理可以进行查看添加与删除，可设置来访人员允许访问的时间段。

5、可通过右上角的扫描二维码图标来扫描二维码通过门禁（适用于人脸门禁未成功识别的情况）。

6、可通过操作门禁端来发送推送通知到APP，并在APP端查看门禁端的实时内容（与门禁处实时视频流交互）。
